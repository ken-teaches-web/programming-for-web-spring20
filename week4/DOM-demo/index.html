DOM Demo

When working with JavaScript, many developers choose to use libraries and frameworks to access and manipulate the DOM (Document Object Model). These libraries make selecting elements easier, as well as manipulating position and style.

While I don't think everyone needs to be a DOM scripter—I certainly don't want to be—it does make sense to at least see how these things happen.
